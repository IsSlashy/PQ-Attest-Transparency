PQ-Attest-Transparency — client-side receipt verifier (WASM)

This page verifies a confidential-inference session receipt entirely in your browser: post-quantum STH signature (SLH-DSA), Merkle inclusion, session binding, and that the build's root matches an independently-trusted one. No server is contacted to decide the verdict; the WASM verifier uses no randomness.

    About the trust model
    

      The receipt proves non-equivocation (the build is in a public,
      append-only log) and HNDL-safe binding (X25519+ML-KEM-768). It does
      not prove the build is honest — only that it cannot be served to you in
      secret. The trusted_root here stands in for a root obtained from
      independent witnesses (milestone M4) or an on-chain anchor.
